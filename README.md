# GRPC-Web

This example shows how to build an angular(or any js-framework web-app) which is use GRPC-WEB to communicate
Including 3 parts:

  - A Go-Server has a sum 2 number service, which is using GRPC
  - An Envoy proxy server
  - An Angular Application uses GRPC-WEB

# How it works

  - Go to every single folder and run
  - README files in each folder may help!
